# IoT-Based Weather Monitoring Station

Real-time weather monitoring system using **Arduino UNO R4 WiFi** and **DHT22 sensor** with cloud dashboard and SD card storage.

## Features
- Temperature & humidity monitoring every 60 seconds
- Live display on built-in 12x8 LED matrix
- Cloud dashboard via ThingSpeak
- SD card local storage (CSV format)
- Offline data buffering (200 readings) for zero data loss
- Fully autonomous - runs on any USB power source

## Hardware
- Arduino UNO R4 WiFi
- DHT22 (AM2302) Sensor
- Micro SD Card Module (SPI)
- USB Power Bank / Charger
